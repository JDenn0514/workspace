### Infrastructure note â€” `R/sgp.R` top-level execution

`devtools::document()` and `devtools::load_all()` fail due to a bare `result <- sgp(...)` call at the end of `R/sgp.R` (lines 709â€“713). This is a pre-existing defect. The workaround used for this run:

```r
my_load <- function(path) {
  env <- new.env(parent = globalenv())
  r_files <- sort(list.files(file.path(path, 'R'), pattern = '[.]R$', full.names = TRUE))
  for (f in r_files) {
    if (grepl('sgp[.]R$', f)) {
      tryCatch(source(f, local = env), error = function(e) invisible(NULL))
    } else {
      source(f, local = env)
    }
  }
  env
}
roxygen2::roxygenise('.', roclets = c('namespace', 'rd'), load_code = my_load)
```
